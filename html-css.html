RELATIVE POSITIONING IN CSS CODE

Relative positioning is where you manipulate your chosen element, in relation to its OWN original/default/static position either in its given 'container' or on the page. when moving an element, you are offsetting FROM whatever original edge position you are wanting to move away from. (+) numbers move inward from the original edge position (towards the centre of the container or page)and (-)numbers move outward from mthe original edge position (towards the outer edges of the container or page)

ABSOLUTE POSITIONING IN CSS CODE

Absolute positioning differs from relative positioning simply by changing the edges that you are "offsetting" from. the edges of your moving element are not used, now the edges of the 'container' or page that your chosen element sits inside of, are what you are moving either toward or away from. (+) numbers position you in from the selected container edges, while (-) numbers can sendyou outside the edges of the container or page.

FIXED POSITIONING IN CSS CODE

Fixed positioning uses 1 set of edges and one set only for you to offset from. And that is the "viewport" or browser window. Think of it like a missing square of mirror glass on your hanging mirror at home. No matter what else you can see in that mirror and no matter if you move mirror somewhere else, that missing square will always be there and you wont see anything else in it.
